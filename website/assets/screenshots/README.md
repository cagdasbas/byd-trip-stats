# Screenshots

Drop your PNG (or WebP/JPG) screenshots into **this folder** using the exact
filenames below. The site shows a labelled empty frame until the matching file
exists, then swaps in the image automatically — no HTML edits required.

Recommended: landscape ~16:10, ≥ 1280px wide, exported as `.png`.

| Filename | Where it appears | Suggested shot |
|---|---|---|
| `hero-dashboard.png`        | Hero (top of page)      | Live dashboard, dark theme — your best-looking screen |
| `og-cover.png`              | Social share preview    | 1200×630 banner (can reuse the hero shot) |
| `range-projection.png`      | Range Engine section    | Live range projection view |
| `dashboard-dark.png`        | Screenshots gallery     | Dashboard, dark theme |
| `dashboard-light.png`       | Screenshots gallery     | Dashboard, light theme |
| `trip-history.png`          | Screenshots gallery     | Trip history list |
| `trip-detail.png`           | Screenshots gallery     | Trip detail / analysis |
| `route-map.png`             | Screenshots gallery     | OpenStreetMap route overlay |
| `charts.png`                | Screenshots gallery     | High-resolution charts |
| `heatmaps.png`              | Screenshots gallery     | Correlation heatmaps |
| `charging-detail.png`       | Screenshots gallery     | Charging session detail |
| `battery-degradation.png`   | Screenshots gallery     | Battery health / SoH trend |

> If you want a different set of gallery images, edit the `data-shot` / `src`
> filenames in `website/index.html` to match what you upload.
